# How to work on this repo

## Getting started

- Go to ```https://wokwi.com/projects/460728034055882753```
- Read ```main_example.py``` for example commands

## Working

- On this repo, see and edit on ```main.py``` of the exercise that you want (for example, ex1.1 -> edit ```main_1.1.py```)
- **Test**: paste content from there to the ```main.py``` file on wokwi and run
